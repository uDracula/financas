Configurar Sincronização

Para sincronizar seus dados entre iPhone e PC, você precisa criar um projeto Firebase gratuito. Faça isso uma vez e nunca mais precisará repetir.

Passo 1

Acesse console.firebase.google.com e clique em "Criar projeto". Dê qualquer nome (ex: minhas-financas) e desative o Google Analytics.

Passo 2

No menu lateral, clique em Firestore Database → Criar banco de dados → escolha Modo de teste → selecione a região southamerica-east1.

Passo 3

Clique em Configurações do projeto (ícone ⚙️) → Seus aplicativos → clique no ícone </> (Web) → registre o app → copie o objeto firebaseConfig.
